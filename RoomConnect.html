RoomConnect

A Video Conferencing App

Project Description

RoomConnect is a video conferencing application built using React.js and Redux. It allows users to create and join virtual rooms where they can conduct video calls with other participants. The app offers features such as real-time chat, screen sharing, and search video calls.

Technical Details

Here are some technical details about RoomConnect:

One of the biggest challenges of building a video conferencing web app is maintaining WebRTC connections. My solution to this challenge is to use custom React hook. React hooks are a way to reuse stateful logic in functional components. By creating custom hooks, I encapsulate the logic needed to maintain WebRTC connections, making it easier to maintain and test the code.

Another challenge I faced was implementing rooms for video conferencing using WebSockets. I needed to ensure that multiple users could join a room and communicate with each other seamlessly. To solve this challenge, I used WebSockets to implement rooms for video conferencing.

Continuous Integration (CI) and Deployment (CD) are essential for ensuring that any changes made to the codebase are thoroughly tested and deployed to production promptly. Setting up CI/CD pipelines can be challenging, but platforms like GitHub Actions can simplify the process. This can help reduce errors and speed up the development process.
